箱线图 ​

概述 ​

箱线图（Boxplot）是一种用于展示数据分布的图表类型，能够直观地显示数据的最小值、Q1（中位数下四分位数）、中位数、Q3（中位数上四分位数）和最大值。

基本用法 ​

tsx
import { BoxplotChart } from '@agions/taroviz';

const App = () => {
  const option = {
    title: {
      text: '箱线图示例',
    },
    xAxis: {
      type: 'category',
      data: ['Group A', 'Group B', 'Group C'],
    },
    yAxis: {
      type: 'value',
    },
    series: [
      {
        type: 'boxplot',
        data: [
          [850, 940, 980, 1050, 1130],  // Group A: min, Q1, median, Q3, max
          [920, 1000, 1050, 1150, 1200], // Group B
          [780, 850, 920, 1050, 1150],   // Group C
        ],
      },
    ],
  };

  return <BoxplotChart option={option} width="100%" height={400} />;
};

API ​

Props ​

属性类型默认值说明
optionBoxplotOption-ECharts 箱线图配置
widthstring | number'100%'图表宽度
heightstring | number400图表高度
classNamestring-CSS 类名
styleCSSProperties-内联样式
loadingbooleanfalse是否显示加载状态
themestring-主题名称

BoxplotOption ​

继承自 ECharts 的 Boxplot Series，主要配置项：

属性类型说明
type'boxplot'图表类型
datanumber[][]箱线图数据，每项为 [min, Q1, median, Q3, max]
namestring系列名称
itemStyleobject箱体样式
emphasisobject高亮样式
