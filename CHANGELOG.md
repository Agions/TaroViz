# 更新日志

本文档记录 TaroViz 的所有重要变更。

## [1.0.0] 

### 重大变更

- 📦 包名更新：主包从 `@agions/taroviz-all` 改为 `@agions/taroviz`

### 新增

- 🎉 首次发布
- 📊 支持基础图表类型：折线图、柱状图、饼图
- 🎨 主题系统支持
- 🔄 数据处理工具
- 📱 多端适配支持

### 核心包更新

#### @agions/taroviz-core@1.0.0

- ✨ 基础图表组件
- 🛠️ 工具函数库
- 📱 多端渲染支持

#### @agions/taroviz-hooks@1.0.0

- 🎯 图表数据处理 Hooks
- 🔄 状态管理 Hooks
- 🎨 主题 Hooks

#### @agions/taroviz-data@1.0.0

- 📊 数据转换工具
- 📈 数据聚合功能
- 🔍 数据过滤与验证

#### @agions/taroviz-themes@1.0.0

- 🎨 预设主题
- 🛠️ 主题创建工具
- 🔄 主题切换支持

#### @agions/taroviz-adapters@1.0.0

- 📱 多端适配器
- 🔌 平台兼容层
- 🛠️ 渲染优化

### 修复

- 修复图表渲染性能问题
- 修复主题切换闪烁问题
- 修复数据更新不及时问题

### 优化

- 优化图表渲染性能
- 优化数据处理效率
- 优化包体积
