TaroViz

基于Taro和ECharts的多端图表组件库

开始使用

什么是 TaroViz?

TaroViz 是一个基于 Taro 和 ECharts 的图表组件库，为 Taro 开发者提供易用、高效的数据可视化解决方案。它支持多种小程序平台和 H5，让你的数据可视化能力更上一层楼。

🚀 多平台支持

支持微信小程序、支付宝小程序、百度小程序等多个平台，以及H5。

📊 丰富的图表类型

提供柱状图、折线图、饼图、散点图等多种图表类型。

🙌 易于使用

简单的API设计，快速上手，轻松集成到Taro应用中。

🎨 高度可定制

提供丰富的配置选项，支持自定义主题和交互。

快速开始

npm install @agions/taroviz

或

yarn add @agions/taroviz

或

pnpm add @agions/taroviz

详细的使用方法，请参考使用指南。

核心功能

  • 丰富的图表类型：支持折线图、柱状图、饼图、散点图、雷达图等多种图表类型
  • 自定义主题：提供内置主题和自定义主题功能，满足不同的设计需求
  • 响应式适配：自动适配不同屏幕尺寸，提供良好的移动端体验
  • 简化的数据处理：提供直观的数据处理工具，轻松处理和转换数据
  • 丰富的交互事件：支持点击、触摸等多种交互事件，增强用户体验
  • TypeScript支持：完整的TypeScript类型定义，提供代码提示和类型检查
